# FITOM_X ネイティブパッチ リファレンス

「ネイティブパッチ」は、通常モード(CC#0=0、バンクセレクトLSB+プログラムチェンジで選ぶ標準的な音色選択方式)で使われる音色データです。1つ以上の「レイヤー」を束ね、レイヤーごとに異なる音源チップ・音色(HwPatch)を鳴らすことで、複数チップを重ねた分厚い音や、鍵盤分割(スプリット)を作れます。

音色そのものの合成パラメータ(FB/ALG/オペレータ設定等)は [HwPatch リファレンス](./hwpatch-reference.md) を、ビブラート・ベロシティ感度等の演奏特性は [SwPatch リファレンス](./swpatch-reference.md) を参照してください。MIDI経由でこれらのパラメータを演奏中に上書きする方法は [MIDI メッセージ リファレンスマニュアル](./midi-message-reference.md) を参照してください。

---

## 1. 全体構成

1つのパッチバンクファイルは、複数のパッチ(プログラム)を持ちます。

```json
{
  "name": "Strings Bank",
  "patches": [
    {
      "prog": 0,
      "name": "Strings 1",
      "sw_bank": 0,
      "sw_prog": 0,
      "poly": 0,
      "layers": [
        {
          "voice_patch_type": 25,
          "hw_bank": 0,
          "hw_prog": 0,
          "note_range_lo": 0,
          "note_range_hi": 127,
          "transpose": 0,
          "volume_offset": 0,
          "pan_offset": 0,
          "enabled": true
        }
      ]
    }
  ]
}
```

## 2. パッチ単位のフィールド

| フィールド | 内容 |
|---|---|
| `prog` | プログラムチェンジ番号(0–127)。このパッチを呼び出すための番号です。 |
| `name` | パッチの表示名。 |
| `poly` | 同時発音数の上限。0(デフォルト)を指定すると、実際に使うデバイスのチャンネル数から自動的に決まります。通常は0のままで問題ありません。 |
| `sw_bank` / `sw_prog` | このパッチ全体で使うパフォーマンスパッチ(演奏特性データ、[SwPatch リファレンス](./swpatch-reference.md)参照)の参照先。省略、または該当するデータが無い場合はパフォーマンスパッチ無し(平坦な特性)として扱われ、無音にはなりません。 |
| `layers` | 下記「3. レイヤー」を1〜4個持つ配列。配列の先頭(インデックス0)がプライマリレイヤーです。 |

## 3. レイヤー(ToneLayer)

1つのレイヤーは「どのチップの、どの音色を、どの音域で、どんな味付けで鳴らすか」を表します。複数のレイヤーを重ねることで、たとえば「OPNのストリングス + DCSGのベースを鍵盤分割で鳴らす」といった構成が作れます。

| フィールド | 範囲 | 内容 |
|---|---|---|
| `voice_patch_type` | 1–255(実際に使うのは1–0x6F) | このレイヤーが使うチップの種別。MIDIの直接デバイス選択値(CC#0)と全く同じ体系です。値の一覧は[MIDIメッセージリファレンスマニュアルの「直接デバイス選択値」表](./midi-message-reference.md#直接デバイス選択値cc0一覧)を参照してください。 |
| `hw_bank` | 0–127 | このレイヤーが参照するHwBank(音色データバンク)の番号。 |
| `hw_prog` | 0–127 | HwBank内のプログラム番号。 |
| `note_range_lo` / `note_range_hi` | 0–127 | このレイヤーが発音するMIDIノート番号の範囲(音域)。デフォルトは全音域(0–127)。鍵盤分割(スプリット)を作る場合、レイヤーごとに異なる範囲を指定します。 |
| `transpose` | -48–+48 | 半音単位のトランスポーズ。 |
| `volume_offset` | -64–+63 | このレイヤーだけの音量オフセット(相対値)。複数レイヤーの音量バランスを調整する際に使います。 |
| `pan_offset` | -64–+63 | このレイヤーだけのパンオフセット(-64=左、0=中央、+63=右)。 |
| `enabled` | true/false | このレイヤーを使用するかどうか。falseにすると、他のフィールドの値に関わらず発音しません。 |

### 音域が重なる場合の扱い

複数のレイヤーの音域(`note_range_lo`〜`note_range_hi`)が重なっている場合、そのノート番号ではすべての該当レイヤーが同時に発音します(排他的な選択ではなく、重ね合わせです)。

### 演奏中のレイヤー上書き

MIDIチャンネルが通常モードでこのパッチを選択している間、NRPN(97番台)を使ってレイヤーの内容(`voice_patch_type`/`hw_bank`/`hw_prog`を含む音色そのものの差し替えや、`transpose`/音域/音量オフセット/パンオフセット)を演奏中に一時的に上書きできます。パッチが元々定義していないレイヤー番号に新たにチップを割り当てることも可能です。詳細は[MIDIメッセージリファレンスマニュアルの「3.3 ToneLayerオーバーライド」](./midi-message-reference.md#33-tonelayerオーバーライドnrpn-97)を参照してください。この上書きは、次にプログラムチェンジを受信するまで有効です。
