# FITOM_X MIDI インプリメンテーションチャート

FITOM_X が受信するMIDIメッセージの対応状況一覧。各メッセージの詳細な仕様は「[MIDI メッセージ リファレンスマニュアル](./midi-message-reference.md)」を参照。

| 機能 | 送信 | 受信 | 備考 |
|---|---|---|---|
| ベーシックチャンネル | × | 1–16 | 全16chが独立してメロディ/リズムいずれの役割にもなれる。固定のドラム専用チャンネルはない(CC#0での動的切替、後述)。 |
| モード | × | モード3・モード4 | 既定はモード3(OMNI OFF・POLY)。CC#126(Mono Mode On)でモード4(OMNI OFF・MONO)に切り替えられる。OMNI ON系のモード(1・2)は非対応。 |
| ノートナンバー<br>　:トゥルーボイス | × | 0–127<br>(音色設定に依存) | 実際に発音される音域は、選択中の音色・デバイスの仕様によって制限される場合がある。 |
| ベロシティ | × | ノートオン：1–127<br>ノートオフ：受信するが値は無視 | ノートオンでベロシティ=0を送るとノートオフとして扱われる(標準MIDI仕様通り)。 |
| アフタータッチ | × | × | キー、チャンネルとも非対応(受信しても無視)。 |
| ピッチベンド | × | ○(14bit) | レンジはRPN#0で設定(半音単位。セント成分の微調整は行われない)。 |

## コントロールチェンジ

| CC# | 送信 | 受信 | 備考 |
|---|---|---|---|
| 0 | × | ○ | バンクセレクトMSB。値により「通常バンク選択」「直接デバイス選択」「チャンネル役割切替」のいずれかとして働く。詳細はメッセージリファレンス参照。 |
| 1 | × | ○ | モジュレーション(ビブラート)デプス。音色側でビブラート機能が有効な場合のみ効果がある。 |
| 2 | × | ○ | ブレスコントローラー。トレモロ(AM)デプスとして扱われる(CC#4と共通)。 |
| 4 | × | ○ | フットコントローラー。トレモロ(AM)デプスとして扱われる(CC#2と共通)。 |
| 5 | × | ○ | ポルタメントタイム。 |
| 6 / 38 | × | ○ | データエントリー MSB / LSB。RPN・NRPNの値設定に使用。 |
| 7 | × | ○ | チャンネルボリューム。0–127。 |
| 10 | × | ○ | パン。0–127(64=中央)。 |
| 11 | × | ○ | エクスプレッション。0–127。 |
| 32 | × | ○ | バンクセレクトLSB。CC#0の値に応じて意味が変わる。 |
| 64 | × | ○ | サステイン(ダンパー)ペダル。0–63=オフ、64–127=オン。 |
| 65 | × | ○ | ポルタメント オン/オフ(64以上でオン)。 |
| 66 | × | ○ | ソステヌート。0–63=オフ、64–127=オン。オンにした瞬間に発音中のノートを保持する。 |
| 68 | × | ○ | レガートフットスイッチ(64以上でオン)。 |
| 84 | × | ○ | ポルタメントコントロール(ソースノート)。次のノートオンのグライド元を指定する。 |
| 96 | × | ○ | データインクリメント。現在選択中のRPN/NRPNの値を1ステップ増加させる。 |
| 97 | × | ○ | データデクリメント。現在選択中のRPN/NRPNの値を1ステップ減少させる。 |
| 98 / 99 | × | ○ | NRPN LSB / MSB。 |
| 100 / 101 | × | ○ | RPN LSB / MSB。 |
| 120 | × | ○ | オールサウンドオフ。 |
| 121 | × | ○ | リセットオールコントローラー。 |
| 123 | × | ○ | オールノートオフ。 |
| 126 | × | ○ | モノモードオン。値(M)によって動作が変わる。詳細はメッセージリファレンス参照。 |
| 127 | × | ○ | ポリモードオン。 |
| 上記以外 | × | × | 受信しても無視される(状態変化なし)。 |

| 機能 | 送信 | 受信 | 備考 |
|---|---|---|---|
| プログラムチェンジ | × | ○ 0–127 | 選択中のバンク設定(CC#0/CC#32)の内容に応じて、実際に呼び出される音色が決まる。 |
| システムエクスクルーシブ | × | △ | Universal Real Time SysExの一部(マスターボリューム/マスターチューニング/スケール・オクターブチューニング)にのみ対応。詳細はメッセージリファレンス参照。GM System On/Off、Identity Requestには対応しない。 |
